 Tech Job Trends — Job Market Analytics Platform
 
 Overview

This project analyzes real-world job postings data to uncover trends in technical skill demand, experience requirements, education levels, hiring locations, and top employers in the tech industry.
The goal is to build an end-to-end data analytics pipeline — from data ingestion and cleaning to NLP-based skill extraction, SQL analytics, dashboards, and predictive modeling.
